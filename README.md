## How to Deploy on GitHub Pages

### Step 1: Create GitHub Repository
1. Go to [github.com](https://github.com) and sign in
2. Click the **+** icon (top right) → **New repository**
3. Name it: `premier-investor-vault`
4. Make it **Public**
5. Click **Create repository**

### Step 2: Upload Files
1. In your new repo, click **"uploading an existing file"**
2. Drag and drop ALL files (HTML, CSS folder, JS folder)
3. Scroll down, type `Initial commit` in the message box
4. Click **Commit changes**

### Step 3: Enable GitHub Pages
1. Go to **Settings** tab (top of repo)
2. Scroll to **Pages** section (left sidebar)
3. Under **Source**, select **Deploy from a branch**
4. Select **main** branch, folder **/(root)**
5. Click **Save**
6. Wait 1-2 minutes, then visit:
https://yourusername.github.io/premier-investor-vault/
